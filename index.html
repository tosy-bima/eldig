01Latar Belakang & Problem Statement

Mengapa Sistem Ini Dibutuhkan?

Sistem kontrol akses merupakan komponen kritis dalam infrastruktur keamanan fisik modern. Kebutuhan akan mekanisme validasi identitas yang andal, deterministik, dan bebas dari ketergantungan pada mikrokontroler memunculkan tantangan desain yang relevan secara akademis maupun industri.

Sistem berbasis logika digital murni — dibangun dari gerbang TTL standar, flip-flop, dan rangkaian sekuensial — menawarkan transparansi operasional penuh, prediktabilitas timing, dan ketahanan terhadap kegagalan perangkat lunak.

Pernyataan Masalah

Bagaimana merancang sistem kontrol akses digital 4-digit yang mampu memvalidasi kombinasi PIN secara sekuensial, mengelola transisi state sistem melalui Finite State Machine (FSM), dan memberikan respons keluaran terstruktur (UNLOCK / LOCK / ERROR) — sepenuhnya menggunakan komponen logika digital fundamental TTL seri 74LS tanpa mikrokontroler?

Batasan Desain

  • Input: Keypad 9 tombol angka (1–9) + tombol Reset; representasi 4-bit BCD
  • Proses validasi: Rangkaian kombinasional + sekuensial berbasis IC 74LS-series
  • Output: Tiga indikator LED (UNLOCK, LOCK, ERROR) + tampilan 7-segment 3 digit
  • Memori status: Flip-flop D (74LS175D); tanpa ROM, RAM, atau EEPROM
  • Simulator: Multisim 14.x (simulasi fungsional penuh)

05Diagram State — Finite State Machine (FSM)

Sistem mengimplementasikan FSM dengan 4 state utama yang dikendalikan oleh flip-flop D (74LS175D). Transisi state digerakkan oleh sinyal CLK dan kondisi logika kombinasional.

S0: IDLE / LOCK S1: DIGIT ENTRY (1–4) S2: UNLOCK S3: ERROR

07Penurunan Persamaan Logika & Karnaugh Map

Persamaan Logika Akhir (Setelah Minimisasi)

Fungsi UNLOCK (Out1):
Out1 = AQ1 • AQ2 • AQ3 • AQ4

Fungsi ERROR / LOCK (Out2):
Out2 = NOT(Out1) • SUBMIT

Next State (D-FF excitation untuk S1):
D_in = (KEY_pressed • NOT(SUBMIT) • S0)
     + (S1 • NOT(SUBMIT) • n<4)

Register Load Enable (per digit i):
EN_i = Decoder_out_i • CLK_pulse

Karnaugh Map — Fungsi Out1 (2 variabel representatif: AQ3, AQ4)

Saat AQ1=AQ2=1 (digit 1 dan 2 cocok), fungsi Out1 bergantung pada AQ3 dan AQ4:

Hasil minimisasi: Out1 = AQ3 • AQ4 (dengan asumsi AQ1=AQ2=1) → persamaan 4-variabel: AND semua.

Total gate count kombinasional: 7 gerbang logika + 4 IC register. Desain menghindari redundansi: fungsi inversi, enable, dan output dikombinasikan secara minimal menggunakan IC TTL seri 74LS yang terintegrasi.

Penggunaan 74LS147D sebagai Priority Encoder mengeliminasi kebutuhan rangkaian encoding manual (9 gerbang AND → 1 IC). Penggunaan 74LS139D sebagai decoder 2-to-4 menggantikan 4 gerbang AND individual untuk pemilihan slot register.

10Kesimpulan & Kontribusi Teknis

Capaian Desain

  • Sistem berhasil mengimplementasikan FSM 4-state menggunakan komponen TTL 74LS-series tanpa mikrokontroler
  • Validasi PIN 4-digit dilakukan secara paralel dan deterministik melalui komparator 721a85
  • Memori state diimplementasikan menggunakan 4 unit 74LS175D dengan clock sinkron dari LM555CM
  • Reset asinkron global memastikan sistem selalu dapat dikembalikan ke state awal

Pemenuhan CLO

  • CLO-3 (C3): Terpenuhi melalui desain rangkaian sekuensial FSM dengan flip-flop D, diagram state, tabel transisi, dan penurunan persamaan eksitasi
  • CLO-4 (C3): Terpenuhi melalui integrasi sistem kombinasional + sekuensial, implementasi Multisim, dan validasi pengujian seluruh skenario fungsional

Limitasi & Pengembangan

  • PIN bersifat hardwired; pengembangan dapat menggunakan EEPROM untuk PIN yang dapat diubah
  • Sistem saat ini single-user; dapat diperluas dengan multiplexing untuk multi-user
  • Penambahan timeout counter (74LS93D) untuk membatasi percobaan salah
  • Implementasi PCB fisik sebagai tahap lanjutan verifikasi hardware
